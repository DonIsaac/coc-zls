# Coc Zig

![Zig Language Server](./.github/assets/zls.svg)

[Coc.nvim](https://github.com/neoclide/coc.nvim) wrapper for [ZLS](https://github.com/zigtools/zls).

## Install

In you Neovim run:

`:CocInstall coc-zig`

Or using Packer:

```lua
use {'UltiRequiem/coc-zig', run='yarn install --frozen-lockfile && yarn build'}
```
